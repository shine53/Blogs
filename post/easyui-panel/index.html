easyui--panel

2019-06-07

其实写下来的这些东西文档中都有相关内容，在写一遍是为了更好的备注

panel 的创建

  1. html
<div id="p" class="easyui-panel" title="My Panel" style="width:500px;height:150px;padding:10px;background:#fafafa;" data-options="iconCls:'icon-save',closable:true, collapsible:true,minimizable:true,maximizable:true">
	<p>panel content.</p>
	<p>panel content.</p>
</div>

常用属性： 1. href: 请求同源页面，只能请求到 body 中的内容（如果 script 写在 head 中，求取不到），如果数据求取不到，会有一个 loading 动画，知道数据加载成功；
3. padding：只对 panel-body 起作用；
4. closable: 显示关闭按钮；
5. miniminzable: 显示缩小按钮，跟 closable 的效果一样，都是隐藏起来，可以在角落写一个小方块，点击时让 panel 再次显示即可；
6. maximizable: 显示最大化按钮，如果设置了 fit: true, 则不起作用；
7. tools：自定义工具组
面板（panel）工具组可通过已存在

标签声明

<div class="easyui-panel" style="width:300px;height:200px" title="My Panel" data-options="iconCls:'icon-ok',tools:'#tt'"></div>
<div id="tt">
		<a href="#" class="icon-add" onclick="javascript:alert('add')"></a>
		<a href="#" class="icon-edit" onclick="javascript:alert('edit')"></a>
</div>

	<!--	这种添加方式，#tt 只能被调用一次 -->
	```  
					面板（panel）工具组可通过数组定义  
```html
<div class="easyui-panel" style="width:300px;height:200px" title="My Panel" data-options="iconCls:'icon-ok',tools:[
	{
		iconCls:'icon-add',
		handler:function(){alert('add')}
	},{
		iconCls:'icon-edit',
		handler:function(){alert('edit')}
	}]">
</div>

这里有事件，不能写在 data 外面
