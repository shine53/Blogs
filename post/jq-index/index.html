2019-06-03

————因为长时间使用框架，很多基本东西有些不习惯了，抽空每次写一个功能，并且简单记录。

常用在联级菜单

index() 方法传递参数，那么返回值就是这个 jQuery 对象集合中第一个元素相对于其同辈元素的位置。 注意：必须是同一个父元素下的 jq 对象集合，否则不起作用。

比如

<div id="nav"> 
	<a href="http://www.51xuediannao.com/">建站素材</a> 
	<a href="http://www.51xuediannao.com/">jquery 特效</a> 
	<a href="http://www.51xuediannao.com/">懒人主机</a> 
	<a href="http://www.51xuediannao.com/qd63/">前端路上</a> 
</div> 

	$("#nav a").click(function(){ 
				//四个经典的用法 
				var index1 = $("#nav a").index(this);  //  我的常用
				var index2 = $("#nav a").index($(this)); 
				var index3 = $(this).index() 
				var index3 = $(this).index("a") 
 });
